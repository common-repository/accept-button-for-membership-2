== Description ==

Simple plugin that adds a shortcode to hide PayPal subscribe button for Membership 2 plugin until user clicks 'I ACCEPT' button.

Add shortcode '[acceptbutton]' to description of the membership to display a 'I ACCEPT' button, which must be pressed to display the PayPal subscribe button.

== Installation ==
1. Upload `accept-button-for-membership-2` to the `wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Insert shortcode in description of membership.
